# Proyecto de Ingeniería de Software II  

## Estructura del repositorio  
- `src/` - Código fuente  
- `doc/` - Documentación  
- `bin/` - Ejecutables  
- `script/` - Scripts auxiliares  

## Instalación  
1. Clonar el repositorio  
2. Instalar dependencias con pip  
3. Ejecutar scripts
